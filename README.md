## Run Locally

Clone the project

```bash
  git clone https://github.com/SFarazH/attendease.git
```

Go to the project directory

```bash
  cd attendease
``` 

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  py manage.py runserver 
```
